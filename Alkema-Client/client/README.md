# Alkema

A Phaser 3 multiplayer browser game built on [Liberated Pixel Cup](https://lpc.opengameart.org) character sprites.

## What's Working

- Real-time multiplayer via WebSocket (Socket.io)
- Dynamic character sprite loading with full LPC animations (walk, idle, slash, spellcast, and more)
- Mobile-responsive portrait layout with touch d-pad controls
- Camera follows local player across a 4096x4096 tile map
- Warm pixel-art UI with Alagard font

## Tech Stack

- **Phaser 3** — Game engine
- **Socket.io** — Real-time multiplayer
- **Vite** — Build tool
